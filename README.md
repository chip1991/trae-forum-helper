# TRAE 中文社区自动浏览助手

自动浏览 TRAE 中文社区新帖子，点赞后继续浏览下一个

## 功能特点

- 自动浏览论坛新帖子
- 自动点赞帖子
- 可自定义浏览时间
- 实时状态显示

## 安装使用

1. 打开 Chrome 浏览器
2. 访问 chrome://extensions/
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择当前文件夹即可

## 使用说明

1. 访问 [forum.trae.cn](https://forum.trae.cn/)
2. 点击插件图标，打开控制面板
3. 点击"开始自动浏览"按钮
4. 插件会自动浏览每个帖子并点赞
5. 处理完所有帖子后自动停止

## 技术栈

- Chrome Extension Manifest V3
- Vanilla JavaScript
